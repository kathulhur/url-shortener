URL Shortener API Documentation



Description

A URL shortener API containing functionality such as shortening URL and retrieving URL information given the code

ikzi.ml make use of this API. Developed by my web developer friend, goriio



API Base

https://short-url-webapp.herokuapp.com/

Shortening a link

/shorten



Example

GET/POST: https://short-url-webapp.herokuapp.com/shorten?url=www.google.com



Response Object

{
    {
    "apiVersion":"0.0.1",
    "data":{
    "urlCode":"dQJken_i",
    "longUrl":"https://www.google.com",
    "shortUrl":"https://short-url-webapp.herokuapp.com/dQJken_i",
    "date":"2022-06-21T14:54:28.525Z"
    }
}


Getting information on a link

/info

This gets the information about the link. This requires a 'code' query parameter having the short url code. Example: https://short-url-webapp.herokuapp.com/c0d3 --> Code: c0d3



Example

GET/POST: https://short-url-webapp.herokuapp.com/info?code=c0d3



Response Object

{
    {
    "apiVersion":"0.0.1",
    "data":{
    "urlCode":"dQJken_i",
    "longUrl":"https://www.google.com",
    "shortUrl":"https://short-url-webapp.herokuapp.com/dQJken_i",
    "date":"2022-06-21T14:54:28.525Z"
    }
}
